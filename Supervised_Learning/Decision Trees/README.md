---
The diabetes data contains 768 rows, and 9 columns. Link to dataset is: [link](https://www.kaggle.com/datasets/mathchi/diabetes-data-set)

- Pregnancies
- Glucose
- BloodPressure
- SkinThickness
- Insulin
- BMI
- DiabetesPedigreeFunction
- Age
- Outcome
