## Data: 
---
The wineQT data contains 1143  rows, and 13 columns. Link to dataset is: [link](https://www.kaggle.com/datasets/yasserh/wine-quality-dataset)

- fixed acidity
- volatile acidity
- citric acid
- residual sugar
- chlorides
- free sulfur dioxide
- total sulfur dioxide
- density
- pH
- sulphates
- alcohol
- quality
- Id
